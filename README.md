# SpheriUnfold
Enhance the visualization of mapping a 3D object onto a 2D surface

## ★★ Instructions ★★
1. WASD to move
2. Arrows to rotate the angle of view
3. QE to adjust focal point
4. FG to adjust visibility
5. P to pause/continue the animation
